# crypto-dashboard3

## Build Setup

```bash
# Clone repository
$ git clone https://github.com/yahyameo/mc2-finance
# Install frontend dependencies
$ cd frontend
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate

# Install backend dependencies
$ cd backend
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev
```
